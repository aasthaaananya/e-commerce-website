# Ecommerce Project

## Overview

A modern e-commerce web application built with React and Vite that provides a smooth and responsive shopping experience. The project focuses on clean UI design, efficient navigation, and scalable frontend architecture while following modern web development practices.

## Features

* Product browsing and catalog management
* Dynamic routing using React Router
* Responsive user interface
* API integration using Axios
* Fast development and optimized builds with Vite
* Component-based architecture
* Automated testing with Vitest and React Testing Library
* Modern React development practices

## Tech Stack

### Frontend

* React 19
* React Router DOM
* Axios
* Day.js

### Development Tools

* Vite
* ESLint
* Vitest
* React Testing Library

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd ecommerce-project
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

## Available Scripts

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Run Tests

```bash
npm run test
```

### Preview Production Build

```bash
npm run preview
```

### Run Linting

```bash
npm run lint
```
